## üöÄ Daily Breakdown

### **Day 1: Python Fundamentals**
- **Topics:** Basic Syntax, Variables (Int, Float, String, Bool), Input/Output, Arithmetic Operators, Logical Operators.
- **Goal:** Write your first script to calculate Body Mass Index (BMI).
- **Resources:** [Python.org Docs](https://docs.python.org/3/tutorial/introduction.html)

### **Day 2: Data Structures & Functions**
- **Topics:** Lists (methods, slicing), Dictionaries (key-value pairs), Tuples, Sets. Writing reusable functions (`def`).
- **Goal:** Build a "Task Manager" CLI tool using dictionaries and lists.

### **Day 3: NumPy (The Foundation of ML)**
- **Topics:** Creating arrays, `shape`, `reshape`, `dot product`, Broadcast logic.
- **Goal:** Perform matrix operations to solve a system of equations.

### **Day 4: Pandas (The Data Workhorse)**
- **Topics:** Loading CSVs, handling Missing Values (`fillna`, `dropna`), `groupby`, `merge`, and `apply`.
- **Goal:** Analyze a "Sales Dataset" to find top-performing regions.

### **Day 5: Data Visualization**
- **Topics:** Line charts, Scatter plots, Histograms, Boxplots. Aesthetics with Seaborn styles.
- **Goal:** Visualize correlations in a "Housing Price" dataset.

### **Day 6: Intro to ML - Linear Regression**
- **Topics:** The ML Pipeline (Data -> Preprocessing -> Model -> Predict). Loss functions (MSE).
- **Goal:** Predict house prices based on square footage using `sklearn.linear_model.LinearRegression`.

### **Day 7: Classification Algorithms**
- **Topics:** Binary vs. Multiclass classification. Intuition of Decision Trees.
- **Goal:** Predict if a passenger survived the Titanic disaster.

### **Day 8: Unsupervised Learning**
- **Topics:** Finding patterns without labels. Elbow method for K-Means.
- **Goal:** Segment customers of a shopping mall into distinct groups based on spending habits.

### **Day 9: Model Evaluation & Optimization**
- **Topics:** Precision, Recall, F1-Score, Confusion Matrix. GridSearchCV for tuning parameters.
- **Goal:** Improve your Day 7 model using Hyperparameter tuning.

### **Day 10: The Grand Finale (Capstone)**
- **Topics:** Bringing it all together.
- **Task:** 
    1. Select a dataset from Kaggle (e.g., Credit Card Fraud or Wine Quality).
    2. Perform EDA (Exploratory Data Analysis).
    3. Preprocess and Split data.
    4. Train 2-3 models and compare performance.
    5. Present findings with visualizations.
